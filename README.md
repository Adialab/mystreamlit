# MyStreamlit

## 說明

這是一個使用 Streamlit 製作的簡單 Demo，用來展示一些常見的 Streamlit 功能。

## 功能

* 資料表
* 互動式圖表
* 輸入框
* 按鈕
* 警告訊息

## 如何使用

1. 開啟網頁 [https://fgtseng-demo.streamlit.app/](https://fgtseng-demo.streamlit.app/)。
2. 使用滑鼠點選各個功能的按鈕來試用。
